About this Website

I created this website as a supplement to my resume. Not only does it show a more detailed look at my previous work, but it gives a quick insight into my design and software engineering preferences. It was implemented long before the LLM takeover, so this website is certified from-scratch, organic, human-writ. It's an original design to show off my versatility. I try to focus on a simple, but attractive design. Like many modern design trends, I prefer flat, material graphics with few colors and big bold images. The colors chosen for this website follow a theme, as all good things do.

Astronaut Blue: #81b1c9

Astronomers Green: #e8f2eb

Cosmic Latte is the average color of all the stars of the universe; only fitting to include it here.

I like to use libraries, they speed up development and are extensively tested for security. Libraries are half the reason I favor Python for my personal projects. For a simple website like this, I only used Bootstrap, FontAwesome, and jQuery.
